Class: GridSearch

Components.GridSearch

new GridSearch()

Allows a dataset to be searched using free text searches. This component utilizes a property expression to determine which field to search on. See Searcher example for property expression samples. If no property is specified, a custom filterFunc can be passed in to include custom filtering logic. Requires to be nested within a Grid component.
Source:
  • fedramp.components/grid-search.component.js, line 44
Example
// Using property expression
<grid-search property="product.name in products" placeholder="Search by Product Name"></grid-search>

// Using custom filter function
<grid-search filter-func="myCustomFilterFunc" placeholder="Search by Product Name or Provider Name"></grid-search>
